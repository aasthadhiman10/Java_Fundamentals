# Java Fundamentals

Welcome to my **Java Fundamentals** repository! This project tracks my journey as I learn Java programming from scratch. It contains beginner-friendly programs demonstrating core concepts like printing outputs and handling user interactions.

## 🚀 Projects Included

### 1. Print Program (`Print_Program.java`)
A classic introduction to Java that demonstrates how to output text to the console.
* **Concepts covered:** Class structure, the `main` method, `System.out.print()`, and `System.out.println()`.

### 2. Input/Output Program (`Input_Output_Program.java`)
An interactive console application that prompts the user for their name and reads the input.
* **Concepts covered:** Importing utility packages (`java.util.*`), working with the `Scanner` class, reading string inputs using `sc.nextLine()`, and string concatenation.

## 🛠️ How to Run the Programs Local Environment

### Prerequisites
Make sure you have the Java Development Kit (JDK) installed on your system.

### Steps
1. Clone this repository or download the source files.
2. Open your terminal or command prompt in the directory containing the files.
3. Compile the Java file using:
   ```bash
   javac Code_File_Name.java
